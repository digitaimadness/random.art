| this bot tweets random pictures from folder, checks if its art      |
| and generates pixiv links if available (powered by saucenao.com)    |
| or link to anime with timecode!                                     |
| + character recognizing (powered by tensorflow)
|                                                                     |
| setup on linux(recommended):                                        |
| (1)type in terminal:                                                |
|   git clone https://github.com/exife/random.art                     |
|   sudo apt-get update && sudo apt-get install python3 python3-pip   |
|   sudo pip3 install pillow requests tweepy                          |
| (2)enter twitter api keys, path to folder with pics in settings.txt |
|   (howto get them: https://gist.github.com/clayadavis/5681622 )     |
| (3)enter api key from https://saucenao.com/user.php?page=search-api |
|   in settings.txt (register there to get the key)                   |
| (4)customize chance of tweeting and frequency in settings.txt       |
| (optional) setup neural network (moeflow.py) by typing in terminal: |
|   sudo pip3 install tensorflow                                      |
|   cd ./random.art                                                   |
|   git clone https://github.com/nagadomi/animeface-2009              |
|   sudo apt-get install ruby libmagickwand-dev                       |
|   sudo gem install rmagick                                          |
|   cd ./animeface-2009 && ./build.sh                                 |
| (5)run with randomart.py                                            |
| (optional) run autofollow.py for marketing purposes                 |
|                                                                     |
| setup on windows:                                                   |
| 1)download and extract to any folder                                |
| github.com/exife/random.art/releases/download/4.0.4/randomart.zip   |
| 2)enter twitter api keys, path to folder with pics in __settings.txt|
|   (howto get them: https://gist.github.com/clayadavis/5681622 )     |
| 3)enter api key from https://saucenao.com/user.php?page=search-api  |
|   in __settings.txt (register there to get the key)                 |
| 4)customize chance of tweeting and frequency in __settings.txt      |
| 5)run with __randomart.exe                                          |
| (optional) run __autofollow.exe for marketing purposes              |
